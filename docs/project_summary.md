# FraudShield-AI Project Summary

FraudShield-AI is a TensorFlow/Keras credit card fraud detection project using a Dense neural network.

The project includes:

- Exploratory data analysis
- Duplicate removal
- Feature scaling
- Stratified train-test splitting
- Baseline model training
- Class-weighted improvement experiments
- Accuracy, precision, recall, F1-score, and confusion matrix evaluation
- Misclassified transaction analysis
- Limitations and next steps

The selected final approach uses moderate class weighting to improve fraud recall while limiting false positive predictions.
